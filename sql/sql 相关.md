
### SQL--Structured Query Language(结构化查询语句)

	SQL：4大查询语句——增删改查

	增-INSERT
	INSERT INTO 表 (字段列表) VALUES(值列表)
	INSERT INTO `user_table` (`ID`, `username`, `password`) VALUES(0, 'blue2', '987654');
	
	
	删-DELETE
	
	改-UPDATE
	
	查-SELECT
	SELECT 什么 FROM 表
	SELECT * FROM `user_table`;
